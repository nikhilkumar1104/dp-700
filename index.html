Pre-exam instructions

Read the rules before starting

Time limit: The default full mock exam is 120 minutes. Timed quiz mode uses 30 minutes.
Navigation: You can move forward and backward between questions using the buttons or question navigator.
Auto-save: Answers are saved locally every 10 seconds and after every selection.
Review: Use Mark for Review to revisit uncertain questions before submission.
Question types: Single choice, multiple choice, case study, scenario analysis, matching, ordering, architecture, pipeline design, and troubleshooting.
Integrity: Browser focus changes are logged as practice integrity events.
Locked exam flow: After starting, you must finish using the Submit Exam button before using Home, Dashboard, Admin, or browser navigation.
Practice feedback: Use Check My Answer to see whether your current answer is correct and why each option is right or wrong.
